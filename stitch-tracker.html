About the Project

Stitch Tracker is the evolution of Stitch Counter, bringing enhanced features and a modern user experience. Built with cross-platform technologies to provide a seamless experience across web and mobile platforms.

Features

  • Enhanced stitch and row tracking
  • Cross-platform compatibility (web and mobile)
  • Modern, intuitive user interface
  • Project management and organization
  • Shared business logic across platforms

Technologies

Kotlin Multiplatform TypeScript HTML/CSS Cross-Platform
